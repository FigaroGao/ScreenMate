About ScreenMate

ScreenMate

A desktop AI assistant with vision understanding, chat, text-to-speech, multi-model switching, and context memory.

Version{{ config.APP_VERSION }}
Environment{{ config.APP_ENV }}
Python3.11
FrameworkFlask + Bootstrap 5
ArchitectureProvider Pattern + Pipeline + MVC
Work Modes
  • Manual Mode
    Screenshot → Vision → Response → TTS (optional). Quick single-shot analysis.
  • Auto Mode
    Continuous observation with context memory and proactive summaries.
  • Agent Mode Coming Soon
    Autonomous task execution with tool use.
